# Shopify Expert Portfolio

This is a single-page React portfolio designed to showcase Shopify services and past work.
It's ready to deploy on Vercel or Netlify.

## Quick start

1. Install dependencies:
```
npm install
```

2. Run locally:
```
npm start
```

3. Build for production:
```
npm run build
```

Deploy the repository to Vercel by connecting your GitHub repo or by uploading the project folder.
